# Reference Docs

This folder holds historical, audit, and exploratory docs that are kept for context but do not define the current Arcana rewrite contract by themselves.

Use current authority first:
- `docs/arcana-v0.md`
- `docs/specs/spec-status.md`
- `docs/rewrite-roadmap.md`
- domain `v1-scope.md` files and `deferred-roadmap.md` ledgers under `docs/specs/**`

Reference material belongs here when it is:
- a MeadowLang carryover kept for historical context
- an audit record of a completed comparison or adoption pass
- a generic future-direction note that is not the active contract

If a reference doc still contains useful deferred direction, that direction must be copied into an authoritative deferred ledger before rewrite work relies on it.
