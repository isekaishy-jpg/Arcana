# Arcana Desktop Workspace

The old canvas-era desktop proof app is retired from this workspace.

The remaining checked-in member is `arcsb_app`, a focused software-buffer smoke target for `arcana_graphics.arcsb`.

Package it with:
- `arcana package --member arcsb_app --target windows-exe`

That stages a normal native bundle and exercises the current `arcana_desktop` + `arcana_graphics.arcsb` path.
